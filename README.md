Writing an MVP for your Backend with Kotlin
=================

Showcasing how to create your Backend MVP using Kotlin.

Getting Started
--------------------

Clone this project, and import it directly in your IntelliJ. You are ready to go.

Prerequisites
--------------------

* [IntelliJ][1]
* [*AMP][2]
